About this crop recommendation system

Based on “Advancing crop recommendation system with supervised machine learning and explainable artificial intelligence”.

Agriculture is critical for food security and rural development, but farmers must choose the right crop under constraints such as soil health, climate, and limited land. Modern machine learning techniques can support these decisions by learning patterns from data covering soil nutrients, meteorological factors, and crop performance.

The underlying research evaluates multiple supervised algorithms and this project uses a Random Forest classifier for multi‑class crop prediction. It is designed for datasets with features such as nitrogen (N), phosphorus (P), potassium (K), soil pH, temperature, humidity, and rainfall, covering crops like rice, coffee, maize, cotton, and more.

To make model decisions transparent, Explainable AI (XAI) methods such as SHAP are used. They highlight which features most influence each recommendation, giving agronomists and farmers confidence in the predictions and helping them understand why a specific crop is suggested under given conditions.

This UI is a prototype front end that can be connected to a backend implementation in Python (Pandas/NumPy for data handling, Random Forest for prediction) and SHAP explanations to deliver fast, accurate, and interpretable crop recommendations at the field level.

Tech stack

Tools used in this project.

  • Python for programming and ML backend.
  • Random Forest for crop recommendation.
  • SHAP for explainable AI (feature contributions).
  • Pandas, NumPy for data manipulation.
  • HTML, CSS, JavaScript for the UI.
